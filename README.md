## Comment lancer le projet ?

À la racine du projet, vous devez lancer le script suivant:

```bash
./play.sh
```

S'il ne s'exécute pas, essayez:

```bash
chmod +x ./play.sh
```

**Assurez-vous que vous le lancez depuis son dossier parent.**
